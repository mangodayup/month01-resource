## 计算机组成

- 硬件
	- 控制器
	- 运算器  
	（控制器+运算器-->CPU)
	- 存储器
	1. 内存RAM：容量小，速度快，临时存储数据
	2. 硬盘HDD：容量大，速度慢，永久存储数据
    - 输入设备
    - 输出设备

- 软件
	- 操作系统
	- 应用软件：为了某种特定的用途而被开发的软件
	- 软件：程序+文档  
      程序是一组计算机能够识别和执行的指令集合。  
	文档是为了便于了解程序所需的说明性资料。
